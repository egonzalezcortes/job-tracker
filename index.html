🔑 How to Get a Google OAuth Client ID

Follow these steps to enable Gmail access:

  1. Go to console.cloud.google.com
  2. Create a new project (or select an existing one)
  3. Go to APIs & Services → Library and enable the Gmail API
  4. Go to APIs & Services → OAuth consent screen
    Set User Type to External, fill in app name & email, save
  5. Go to APIs & Services → Credentials → Create Credentials → OAuth Client ID
  6. Application type: Web application
  7. Under Authorized JavaScript origins, add the URL where you're running this app
  8. Copy the Client ID and paste it in the field above
  9. Under Test users (OAuth consent screen), add your Gmail address

⚠ This app only requests read-only access to your Gmail. It never stores or sends your emails anywhere.
